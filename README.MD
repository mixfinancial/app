## Installation


1. Install Flask 


    sudo pip install flask
    
    
2. Install required libraries


    sudo pip install -r requirements.txt


3. Install MySQL


    sudo pip install mysql


4. Install SQLAlchemy 


    sudo pip install sqlalchemy
